# 🏛️ Mercado Competitivo & O Papel do Estado na Economia

Dashboard educacional interativo para estudo de **Economia do Setor Público**, **Teoria do Consumidor**, **Teoria da Firma** e **Equilíbrio Geral**.

## 📊 Funcionalidades

| Tab | Conteúdo |
|-----|----------|
| 🏠 Visão Geral | Mapa conceitual das razões para intervenção estatal |
| 🛒 Teoria do Consumidor | Maximização de utilidade, restrição orçamentária, curvas de indiferença, TMS |
| 🏭 Teoria da Firma | Minimização de custo, isoquantas, isocusto, TMST |
| ⚡ Equilíbrio & 1º Teorema | CPP, formas de organização, equilíbrio geral |
| 📊 Eficiência de Pareto | Caixa de Edgeworth interativa, curva de contrato |
| 📉 Monopólio vs Concorrência | Comparação de mercados, peso morto (DWL) |
| 🌍 Bens Públicos & Externalidades | Classificação de bens, Pigou, Tragédia dos Comuns |
| ⚖️ Funções de Musgrave | Alocativa, Distributiva, Estabilizadora |
| 🧠 Quiz | 15 questões de múltipla escolha com feedback |

## 🚀 Como rodar localmente

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 🛠️ Stack

- **Streamlit** — interface interativa
- **Plotly** — gráficos dinâmicos
- **NumPy / Pandas** — cálculos e dados
